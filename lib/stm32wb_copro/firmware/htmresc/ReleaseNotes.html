STM32CubeWB Firmware Package V1.17.0 / 7-June-2023

Main Changes

Maintenance Release for BLE, 802.15.4, Thread and Zigbee updates

  • THREAD updates
    • SED role is now supported for FTD.
    • SSED role is now supported for FTD and MTD.
    • Improved global stability and robustness.
    • Improved security.
  • Zigbee updates
    • - Robustness improvement in case of radio noisy environment
  • BLE updates
    • Compliancy TCRL 2022-2.
    • ACI BLE primitives naming alignment to standard.
    • HCI_LE_ENHANCED_CONNECTION_COMPLETE_EVENT is now automatically used instead of HCI_LE_CONNECTION_COMPLETE_EVENT when advertising extensions are used.
    • Additional check on HSE Startup time.
  • MAC 802.15.4 updates
    • Improvement of radio robustness over heavy overloaded and noisy environment.
  • BLE/MAC Static updates
    • Improvement of radio robustness over heavy overloaded and noisy environment.
  • BLE/Zigbee Dynamic updates
    • Bug corrected in Dynamic Concurrent applications to prevent switching from 802.15.4 to BLE while radio is in an incorrect state.

    • Maintenance Release for CMSIS, HAL/LL Drivers and Projects updates

  • HAL/LL Drivers updates
    • Update HAL/LL drivers to include latest corrections
      • - Update of HAL CRYP, Generic, I2C, SMBUS, TSC, UART, SPI drivers.
      • - Update of LL USB drivers
    • Refer to release notes for further details
  • Projects  updates
    • Update of FW projects following changes in latest version of HAL/LL and Middlewares and alignments with CubeMx
      • - /B-WB1M-WPAN1/Examples/BSP/BSP_Example
      • - /STM32WB5MM-DK/Examples/BSP/BSP_Example
      • - /NUCLEO-WB15CC/Examples/FLASH/FLASH_EraseProgram
      • - /NUCLEO-WB15CC/Examples/CRYP/CRYP_DMA
      • - /P-NUCLEO-WB55.Nucleo/Examples/CRYP/CRYP_DMA
